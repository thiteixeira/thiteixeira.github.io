Useful Tools for Networking Research

This page is my brief attempt to summarize tools used in network research, with examples.
Since there are several tools out there, I'm constantly updating this page.

Networking Tools

Ping

Send ICMP ECHO_REQUEST to network hosts.

Traceroute

Print the route packets trace to network host.

Iperf

Perform network throughput tests.

Server side Client side Result on the server side

Netstat

Print network connections, routing tables, interface statistics, masquerade connections, and multicast memberships.

Tcpdump

Dump traffic on a network.

Tshark

Dump and analyze network traffic.

Dig

DNS lookup utility.

Whois

client for the whois directory service. whois searches for an object in a RFC 3912 database.
